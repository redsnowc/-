# ajax DEMO

基于 Flask 简单表单验证 Demo，需要 python 3.7x 版本支持。

## 使用

```shell script
# 如果没有 pipenv 需要先安装，或者查看 Pipfile 文件自行安装依赖
pip install pipenv
cd demoAjax
pipenv install
pipenv shell
export FLASK_APP=app.py
export FLASK_ENV=development
flask run
# 最后复制本地运行地址到浏览器即可正常查看
```
